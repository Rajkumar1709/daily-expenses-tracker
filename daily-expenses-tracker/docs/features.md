# Features & Overview

The **Daily Expenses Tracker** is an offline-first Progressive Web App (PWA) designed to help you track your finances without needing an internet connection.

## Key Features

### 1. Offline Capability
The app works entirely offline. All data is stored locally on your device using `StorageService` (localStorage). You can add expenses, view charts, and manage your account without Wi-Fi.

### 2. Installable App (PWA)
You can install this website as a native app on your phone or desktop.
- **Mobile**: Tap "Add to Home Screen".
- **Desktop**: Click the install icon in the address bar.

![Install Prompt](../screenshots/install_prompt.png)
*(Screenshot: Browser install prompt)*

### 3. Dashboard
The dashboard provides a quick summary of your financial health.
- **Total Balance**: Real-time calculation of Income - Expenses.
- **Recent Transactions**: List of your latest activity.
- **Quick Actions**: FAB (Floating Action Button) to add transactions.

![Dashboard Screenshot](../screenshots/dashboard.png)
*(Screenshot: Dashboard view with balance and list)*

### 4. Expense & Income Tracking
Easily record your financial activities.
- **Categories**: Select from predefined categories (Food, Travel, Salary, etc.).
- **Payment Modes**: Track Cash, UPI, Card payments.
- **Date & Notes**: Add specific details to each transaction.

![Add Expense Screenshot](../screenshots/add_expense.png)
*(Screenshot: Add Expense form)*

### 5. Smart Analytics
Visualize your spending habits.
- **Pie Chart**: Breaks down expenses by category.
- **Expense Only**: Intelligently ignores income to show true spending distribution.

![Analytics Screenshot](../screenshots/analytics.png)
*(Screenshot: Analytics page with pie chart)*

### 6. Dark Mode & Theming
The app respects your system's theme preference and features a modern UI with glassmorphism effects.
